Apa itu JAVA ?

Java adalah bahasa pemrograman dan platform komputasi yang pertama kali dirilis oleh Sun Microsystems pada tahun 1995. Java telah berkembang dari awal yang sederhana hingga menggerakkan sebagian besar dunia digital saat ini, dengan menyediakan platform yang andal di mana banyak layanan dan aplikasi dibangun. Produk baru dan inovatif serta layanan digital yang dirancang untuk masa depan juga terus mengandalkan Java.


Mengapa menggunakan JAVA ?

  • - Java bekerja pada platform yang berbeda (Windowns, Mac, Linux, Raspbery Pi, dll
  • - Ini adalah salah satu bahasa pemrograman populer di dunia
  • - Ini memiliki permintaan besar di pasar kerja saat ini
  • - Mudah dipelajari dan mudah digunakan
  • - Ini open-source dan gratis
  • - Aman, cepat dan kuat
  • - Bahasa yang berorientasi objek yang memberikan struktur yang jelas pada program dan memungkinkan kode untuk digunakan kembali
  • - Java dekat dengan C++ dan C#, memudahkan pemrograman untuk beralih ke Java atau sebaliknya

Syntax Standar JAVA

Main.java
public class Main {
  public static void main(String[] args) {
   System.out.println("Hello World");
 }
}

Komentar Java

Komentar dapat digunakan untuk menjelaskan kode Java, dan membuatnya lebih mudah dibaca. Itu juga dapat digunakan untuk mencegah eksekusi saat menguji kode alternatif.


Komentar Baris Tunggal

Komentar satu baris dimulai dengan dua garis miring ( // ).

Teks apa pun di antara // dan di akhir baris akan diabaikan oleh Java (tidak akan dieksekusi).

Contoh ini menggunakan komentar baris tunggal sebelum baris kode:

Contoh
// Ini adalah komen
System.out.println("Hello World");
Coba

Contoh ini menggunakan komentar baris tunggal di akhir baris kode:

Contoh
System.out.println("Hello World") // Ini adalah komen;
Coba

Komentar Multi Baris

Komentar multi-baris dimulai dengan /*dan diakhiri dengan */.

Teks apa pun di antara /*dan */akan diabaikan oleh Java.

Contoh ini menggunakan komentar multi-baris (blok komentar):

Contoh
/* Kode di bawah ini akan menampilkan tulisan Hello World
ke layar, dan itu luar biasa */
System.out.println("Hello World");

Tipe Data Primitif

Java adalah bahasa yang diketik dengan kuat. Ini berarti bahwa setiap variabel harus memiliki tipe yang dideklarasikan. Ada delapan tipe primitif di Java, yaitu:

Tipe Data Ukuran Deskripsi
byte 1 byte Menyimpan bilangan bulat dari -128 hingga 127
short 2 byte Menyimpan bilangan bulat dari -32.768 hingga 32.767
int 4 byte Menyimpan bilangan bulat dari -2.147.483.648 hingga 2.147.483.647
long 8 byte Menyimpan bilangan bulat dari -9.223.372.036.854.775.808 hingga 9.223.372.036.854.775.807
float 4 byte Menyimpan bilangan pecahan. Cukup untuk menyimpan 6 hingga 7 digit desimal
double 8 byte Menyimpan bilangan pecahan. Cukup untuk menyimpan 15 digit desimal
boolean 1 bit Menyimpan nilai benar atau salah
char 2 byte Menyimpan satu karakter/huruf atau nilai ASCII

Tipe 1 : Byte

Ini dapat digunakan sebagai pengganti int atau tipe integer lainnya untuk menghemat memori ketika anda yakin bahwa nilainya berada dalam -128 dan 127

Syntax
byte myNum = 100;
System.out.println(myNum);
Coba

Tipe 2 : Short

Syntax
short myNum = 5000;
System.out.println(myNum);
Coba

Tipe 3 : Int

Tipe data int adalah tipe data yang lebih disukai saat kita membuat variabel dengan nilai numerik.

Syntax
int myNum = 100000;
System.out.println(myNum);
Coba

Tipe 4 : Long

Ini digunakan ketika int tidak cukup besar untuk menyimpan nilai. Perhatikan bahwa Anda harus mengakhiri nilai dengan "L"

Syntax
int myNum = 15000000000L;
System.out.println(myNum);
Coba

Tipe 5 dan 6 : Float dan Double

Anda harus menggunakan tipe floatk apan pun anda membutuhkan angka dengan desimal, seperti 9,99 atau 3,14515.
Tipe data float dan double dapat menyimpan bilangan pecahan. Perhatikan bahwa Anda harus mengakhiri nilai dengan "f" untuk float dan "d" untuk double:

Syntax Float
float myNum = 5.75F;
System.out.println(myNum);
Coba
Syntax Double
double myNum = 19.99D;
System.out.println(myNum);
Coba

Tipe 7 : Boolean

Ini digunakan ketika int tidak cukup besar untuk menyimpan nilai. Perhatikan bahwa Anda harus mengakhiri nilai dengan "L"

Syntax
boolean isJavaFun = true;
boolean isFishTasty = false;
System.out.println(isJavaFun);
System.out.println(isFishTasty);
Coba

Tipe 8 : Char

Tipe data char digunakan untuk menyimpan satu karakter. Karakter harus diapit oleh tanda kutip tunggal, seperti 'A' atau 'c':

Syntax
char myGrade = 'B';
System.out.println(myGrade);
Coba
