Apa itu JAVA ?

Java adalah bahasa pemrograman dan platform komputasi yang pertama kali dirilis oleh Sun Microsystems pada tahun 1995. Java telah berkembang dari awal yang sederhana hingga menggerakkan sebagian besar dunia digital saat ini, dengan menyediakan platform yang andal di mana banyak layanan dan aplikasi dibangun. Produk baru dan inovatif serta layanan digital yang dirancang untuk masa depan juga terus mengandalkan Java.


Mengapa menggunakan JAVA ?

  • - Java bekerja pada platform yang berbeda (Windowns, Mac, Linux, Raspbery Pi, dll
  • - Ini adalah salah satu bahasa pemrograman populer di dunia
  • - Ini memiliki permintaan besar di pasar kerja saat ini
  • - Mudah dipelajari dan mudah digunakan
  • - Ini open-source dan gratis
  • - Aman, cepat dan kuat
  • - Bahasa yang berorientasi objek yang memberikan struktur yang jelas pada program dan memungkinkan kode untuk digunakan kembali
  • - Java dekat dengan C++ dan C#, memudahkan pemrograman untuk beralih ke Java atau sebaliknya
